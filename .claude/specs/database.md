# Database Spec

Provider: Supabase (Postgres + pgvector)
Extension required: `CREATE EXTENSION IF NOT EXISTS vector;`

---

## Tables

### `logs`
What the user did ‚Äî objective record.

| Column | Type | Notes |
|--------|------|-------|
| `id` | uuid PK | `gen_random_uuid()` |
| `type` | text | `book \| idea \| class \| connection \| article` |
| `title` | text NOT NULL | Book title, idea name, etc. |
| `author` | text | Optional |
| `chapter` | text | Optional |
| `area` | text | Topic area (e.g. "productivity", "psychology") |
| `created_at` | timestamptz | `now()` |

### `insights`
What the user learned ‚Äî distilled knowledge with vector embedding.

| Column | Type | Notes |
|--------|------|-------|
| `id` | uuid PK | |
| `log_id` | uuid FK ‚Üí `logs.id` | ON DELETE CASCADE |
| `content` | text NOT NULL | Insight in user's own words |
| `area` | text | Denormalized from log for quick access |
| `embedding` | vector(1536) | OpenAI text-embedding-3-small |
| `next_review` | timestamptz | For spaced repetition (Week 5) |
| `ease_factor` | float | SM-2 algorithm factor, default 2.5 |
| `created_at` | timestamptz | |

Index: `ivfflat (embedding vector_cosine_ops) WITH (lists = 10)`

### `connections`
Graph edges ‚Äî relationships between insights.

| Column | Type | Notes |
|--------|------|-------|
| `id` | uuid PK | |
| `source_id` | uuid FK ‚Üí `insights.id` | ON DELETE CASCADE |
| `target_id` | uuid FK ‚Üí `insights.id` | ON DELETE CASCADE |
| `label` | text | Generated by Claude Haiku (5 words or fewer) |
| `strength` | float | Cosine similarity score, default 0.5 |
| `confirmed` | bool | User-confirmed connection, default false |
| `created_at` | timestamptz | |

Unique constraint: `(source_id, target_id)`

## RPC Functions

### `match_insights`
pgvector similarity search ‚Äî used by connection detection background task.

```sql
CREATE OR REPLACE FUNCTION match_insights(
  query_embedding vector(1536),
  match_threshold  float,
  match_count      int,
  exclude_id       uuid
)
RETURNS TABLE (id uuid, content text, similarity float)
LANGUAGE sql STABLE AS $$
  SELECT id, content, 1 - (embedding <=> query_embedding) AS similarity
  FROM insights
  WHERE id != exclude_id
    AND embedding IS NOT NULL
    AND 1 - (embedding <=> query_embedding) > match_threshold
  ORDER BY embedding <=> query_embedding
  LIMIT match_count;
$$;
```

---

## Embedding Model

- **Model:** `text-embedding-3-small` (OpenAI)
- **Dimensions:** 1536
- **Used for:** insights (content), topic cards (title + description)
- **Similarity metric:** cosine ‚Äî stored as `<=>` operator in pgvector
